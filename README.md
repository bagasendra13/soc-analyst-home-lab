# SOC Analyst Home Lab
A practical cybersecurity home lab built to develop SOC Analyst skills through hands-on experience with Windows event logs, Sysmon monitoring, process investigation, and network traffic analysis.

## Lab Environment
- Oracle VirtualBox
- Ubuntu Server
- Windows 10
- Sysmon
- Event Viewer

## Learning Objectives
- Monitor endpoint activities
- Analyze Sysmon events
- Investigate process creation events
- Investigate network connection events
- Build incident investigation workflows
